This plugin shortens every url in your post to a shorter url which makes it easy to remember.

== Description ==

It helps make email-friendly links. This plugin shorten every link present in the post to a very small/tiny url which is always easy to remember. With the help of this plugin you can watch how many people clicked your URL through stats. To view the stats just add "-" sign (minus) to the end of your shortened url.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `short-url.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= Do i need to put any code around the hyperlink =

No you dont have to do anything just hyperlink your URLs the normal way (<a href) 

= My URL's are not getting shortened. Why? =

This might be possible if your hyperlink have a different format. This means that after <a - href should come. It wont shorten url which have <a target="xyz" href="abc">ABC</a> instead of <a href="abc" target="xyz">ABC</a>
